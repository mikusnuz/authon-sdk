# DEPRECATED

This SDK is no longer maintained. Authon is a frontend-only authentication platform.

Use the frontend SDKs instead:
- @authon/js (vanilla JavaScript)
- @authon/react (React)
- @authon/nextjs (Next.js)
- @authon/vue (Vue 3)
- @authon/nuxt (Nuxt 3)
- @authon/svelte (Svelte)
- @authon/angular (Angular)
- @authon/react-native (React Native)

Documentation: https://authon.dev/docs
